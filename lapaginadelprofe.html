Instrucciones para el Repositorio Dev0125

  1. Clona o crea un Fork del repositorio: https://github.com/cencaladev/Dev0125
  2. Abre la carpeta en Visual Studio Code.
  3. Realiza al menos los siguientes tres pasos:
    • Crea un nuevo archivo en formato .html en la raíz del sitio como nueva URL. (Recuerda usar nombres sin espacios ni caracteres especiales).
    • En la página principal "index.html", crea un nuevo item <li> dentro del listado numerado <ol> con un enlace o vínculo <a> con el parámetro href dirigido al archivo html creado anteriormente.
    • Crea un contenido HTML cualquiera en el que quieras mostrar tus habilidades y recuerda incluir un enlace de regreso al inicio.
  4. Una vez terminado, crea un Pull Request con tus cambios al repositorio inicial. (Recomendación: utiliza GitHub Desktop).
